실시간 화상 교육
폴라린서비스 커스터마이징이 가능한 화상 커뮤니케이션 솔루션

  • automatic recording 자동 녹취

    학습 참여자들의 대화를 실시간 자막으로 제공 나의 강의실을 통해 녹취 파일 다운로드 가능

  • real-time recording 실시간 녹화

    설정 한번으로 간편하게 강의 전체 녹화 가능 복습을 위한 강의 전체 영상 다운로드 가능

  • Automatic image quality control 자동 화질 제어

    Amazon Voice Focus 기반 머신러닝을 적용한 에코, 배경 소음제거 네트워크 속도 자동 제어를 통해 끊김 없는 실시간 글로벌 화상교육 지원 (별도 세팅 없이도 AI로 화상 수업 영상 품질 컨트롤)

  • Learning Content Sharing 선명한 학습 콘텐츠 공유

    화면을 단순 고유하는 것이 아닌 Embed 형식으로 공유하여 또렷하게 학습자료 이용 가능
